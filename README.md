## Инструкция по запуску

1. **Распаковать проект** на устройство или клонировать репозиторий.
2. **Настроить конфигурацию:**
   * В корневой папке проекта найдите файл `config.example.php`.
   * Переименуйте его в `config.php`.
   * Откройте файл и укажите свои данные в полях `login` и `password`.
3. **Запустить скрипт:** 
   * Проект готов к работе. При необходимости дополнительные параметры и данные запроса можно изменить в файле `payload.php`.
